# demo-data

Seed PDFs for manual testing and the eventual recruiter demo, generated by
`generate_demo_pdfs.py` (requires PyMuPDF).

These files are **not used by anything today**. There is no upload
endpoint yet -- that's Milestone 3 (Document Ingestion). They are checked
in ahead of time so that milestone can start testing against realistic
content on day one instead of writing a fixture generator under time
pressure. `generate_demo_pdfs.py` is not run by any test or CI job in
Milestone 1.
